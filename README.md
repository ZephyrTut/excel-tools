# excel-tools

基于 **Electron + Node.js + ExcelJS** 的可扩展 Excel 自动化桌面平台。

> 仓库名称已统一为 `excel-tools`（项目标识、文档标题、`package.json` 名称均已更新）。

## 当前阶段

- MVP：按规则拆分 Excel（多 sheet、保留标题、保留常见格式、后台执行、可见日志）。
- 预留：合并汇总、去重、对账、格式转换、批量处理。

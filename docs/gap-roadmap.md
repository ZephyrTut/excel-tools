# 缺口修复清单（P0 / P1 / P2）

## P0（本次已实现）
- 使用 Worker Thread 执行拆分任务，避免主进程长时间阻塞。
- 规则加载增加校验与归一化。
- 支持 `valueTransform`、`renameMap`、`skipEmpty`。
- 修复输出冲突策略：仅在同名文件已存在且不覆盖时加时间戳。
- 拆分列解析支持多字母（如 `AA`）。

## P1（建议下一迭代）
- 增加 `splitBy` 扩展（公式值/显示文本/自定义表达式）。
- UI 增加规则编辑器（sheetRule 增删改查）并可保存到 JSON。
- 更细粒度错误码和用户提示（权限、占用、缺 sheet/列）。
- 增加进度状态（总键数、已完成键数、失败统计）。

## P2（中长期）
- Vue3 + Element Plus 重构渲染层。
- 增加自动化测试与 CI（单元+集成）。
- 扩展格式复制能力（冻结窗格、数据验证、条件格式）。
- 新增 merge/dedupe/reconcile 插件化能力。

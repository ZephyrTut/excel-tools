# Triage Labels

Skills 使用五个 canonical triage roles。这个文件把这些 roles 映射到此 repo 实际使用的 GitHub label 字符串。

| Label in mattpocock/skills | Label in our tracker | 含义 |
| -------------------------- | -------------------- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 Issue |
| `needs-info` | `needs-info` | 等待报告人补充更多信息 |
| `ready-for-agent` | `ready-for-agent` | 描述完整，AI 可直接接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不做 |

当某个 skill 提到 role（例如 "apply the AFK-ready triage label"）时，使用此表中对应的 label 字符串。

如需修改标签名，编辑右侧列即可。

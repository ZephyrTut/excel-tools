# 项目关键决策（V1）

## 已确认技术决策

1. 桌面端框架：**Electron**
2. Excel 引擎：**ExcelJS**
3. 配置格式：**JSON 本地文件**
4. UI 技术：**Vue 3 + Element Plus**
5. 后台执行：**Worker Thread + IPC 日志回传**

## 默认行为决策

- 输出目录：优先用户选择；未选择时使用 `config.defaultOutputDir`
- 命名规则：默认文件名为拆分键，自动清理 Windows 非法字符
- 重名处理：默认不覆盖，自动追加时间戳
- sheet 顺序：默认保持源文件顺序
- 空拆分键：默认跳过（可由规则覆盖）

## 实现约束

- UI、业务逻辑、Excel 引擎严格分层
- 不允许硬编码 sheet 名称/标题行/拆分列
- 异常要可追踪，不允许静默失败
